# Sample Release — Midnight Signal

This is a demo release you can load into Capsule Desktop to try the full workflow without needing your own files or wallet.

## What's included

- **demo-draft.json** — A pre-filled Studio Mode draft for a fictional 3-track EP called "Midnight Signal" by Luna Vex. Includes a collaborator (Kai Drift, producer) and a bonus-track collector benefit.

## How to use it

1. Open Capsule Desktop
2. In Studio Mode, click **Load Draft** on the Release page
3. Select `demo-draft.json` from this folder
4. Walk through each step: Release → Collector Benefit → Review → Publish

## What you'll need to publish

To actually publish (mint on XRPL Testnet), you'll need a wallet credentials file with funded testnet accounts. See the handbook for how to set up testnet wallets.

Without a wallet file, you can still explore the full Studio Mode flow up to the Publish step, review how the release is structured, and see how the readiness checklist works.

## This is a preview

Capsule Desktop is in public preview. This sample helps you understand the workflow before using your own creative work. If something feels confusing or broken, please file an issue — that's exactly the feedback we need.
